# フリーランス税金シミュレーター × AI対話

フリーランス/個人事業主の「**結局いくら残る? いくら取られる?**」に即答する無料ツールです。
所得税・住民税・国民健康保険・国民年金・個人事業税・消費税と**手取り**をまるごとザックリ概算し、
試算結果を文脈に持たせたまま AI に「もし◯◯したら?」を自然な言葉で質問できます。

> 走るフリーランスエンジニアが、自分の不便を AI で潰すついでに公開した個人開発ツールです。
> 戦略・発信の母艦リポジトリ [`start-up-note2`](https://github.com/hiroaki0117/start-up-note2) の
> スピアヘッド(集客マグネット)として開発しています。

## できること

- **入力は最小限**:年間売上・経費・申告区分(青色/白色)・消費税区分・社会保険・扶養。
- **出力はまるごと**:各税額/社会保険、手取り、売上に対する負担率、そして
  **毎月いくら積み立てておけば安心か**。
- **AI対話(差別化の核)**:`simulate_tax` ツールで AI が**実際に再計算**してから
  what-if(「外注費を50万増やしたら?」「2割特例にしたら?」など)に即答します。

## 技術スタック

- Next.js 16(App Router) / React 19 / TypeScript
- Tailwind CSS 4
- AI 対話: Anthropic Claude(`@anthropic-ai/sdk`、既定は安価・高速な Haiku)
- テスト: Vitest

計算ロジックはクライアント/エッジで完結するため**原価ほぼゼロ**。LLM は対話部分のみで使い、
モデルと利用上限を絞って「使い放題で赤字」を回避しています。

## セットアップ

```bash
npm install
cp .env.example .env.local   # AI対話を使う場合は ANTHROPIC_API_KEY を設定
npm run dev                  # http://localhost:3000
```

`ANTHROPIC_API_KEY` 未設定でも、シミュレーター本体(計算・表示)はそのまま動作します
(AI対話のみ準備中メッセージになります)。

## 計算の前提(ざっくり概算)

v1 は「**ざっくり当てる**」方針です。令和6年度(2024年)の制度をベースに、細かい特例は省いています。

- **事業所得** = 売上 − 経費 − 青色申告特別控除(65万/10万/0)
- **所得税** = 課税所得に速算表を適用 + 復興特別所得税(2.1%)
- **住民税** = 所得割10% + 均等割・森林環境税(5,000円)。調整控除は省略
- **健康保険** = 国保(全国的な目安の料率。所得割 + 均等割、賦課限度額でキャップ)/
  任意継続 / その他(国保組合など)/ 扶養内 から選択。任意継続・その他は
  **保険料の実額(年額)を手入力**します。**国保の料率は自治体ごとに大きく異なります**
- **国民年金** = 本人分のみ(令和6年度 月16,980円)。扶養内(第3号)は0
- **個人事業税** = (利益 − 事業主控除290万円)× 5%。
  **対象/対象外をトグルで選択**できます(エンジニア等の非該当業種は対象外で0円)
- **売上・経費の入力** = それぞれ 年額 / 月額×12 / 12ヶ月を個別入力 の3通りから選べます
- **ふるさと納税の上限目安** = 住民税所得割 × 20% ÷ (90% − 所得税率×1.021) + 2,000円(実質負担2,000円で済む寄附のおおよその上限)
- **計算根拠** = 結果の各項目はタップ/クリックで計算式と内訳を確認できます
- **消費税** = 免税/2割特例/簡易課税(サービス業50%)/本則課税(概算)。売上は税込前提
- 売上に対する負担率 = (税 + 社会保険) ÷ 売上

定数は [`src/lib/tax/constants.ts`](src/lib/tax/constants.ts) に年度別にまとめてあり、毎年更新する前提です。
ロジックの検算は [`src/lib/tax/calculator.test.ts`](src/lib/tax/calculator.test.ts) を参照してください。

## プライバシー

入力内容はサーバーに保存しません。AI対話では、回答精度のために現在の試算結果(金額)を
その都度モデルに渡しますが、個人を特定する情報は扱いません。

## 免責

本ツールはあくまで概算・参考値です。正確な税額は確定申告や税理士にご確認ください。
